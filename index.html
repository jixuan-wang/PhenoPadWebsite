For every hour that physicians spend on direct patient care, they spend 2 hours on administrative tasks and EHR documentations. This causes growing dissatisfaction and burnout amount physicians, decreasing the quality of care that they are able to provide to patients.

The goal of the PhenoPad project is to provide convenient and efficient clinical note taking during physician-patient interactions through the use of AI.

PhenoPad is an application that is easy to use and allows clinicians to capture patient notes in realtime. It provides AI-based note taking services which improve the quality of patient-physician interaction by :

Hand Writing Recognition
Captures written scripts, abbreviation detection and disambiguation using deep learning and medical language model.
Speech Recognition
Speech audio processing using medical language model and identifies different speakers using deep learning approach.
Natural Language Processing
Text mining on collected note data for medical term recognition.
Clinical Decision Support
Provides differential diagnosis on diseases and phenotype suggestions based on local context.
